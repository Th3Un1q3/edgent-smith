# Reference: MCP Server Catalog

A catalogue of common MCP servers available for orchestration via `code_mode`.

| Server Name | Title | Description | Key Tools |
|---|---|---|---|
| `tavily` | Tavily Search, Extract, Research | Real-time web search and intelligent data extraction from web pages. | `tavily_search`, `tavily_extract`, `tavily_research` |
| `deepwiki` | DeepWiki - Repository insights | Explore GitHub repositories to learn about libraries, frameworks, and tools. | `askQuestion` |
| `context7` | Context7 - Documentation | Access documentation for libraries, websites, and software. | `query-docs`, `resolve-library-id` |

*Note: Always use `mcp_find` to check for the latest available servers in your specific environment.*
