# DevContainer Commands
Common commands to interact with the environment and its primary services.

- **Verify Runtimes**: `python --version` (3.14) and `node -v` (24).
- **Ollama Check**: `ollama list` to view available models.
- **Trace Query**: `curl http://localhost:16686` to check Jaeger UI availability.

Example:
```bash
python --version
```

Reference: .devcontainer/devcontainer.json
